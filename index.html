Installation for macOS

Install with Homebrew (Recommended)

~ $ brew tap floatpane/floatpane
~ $ brew install floatpane

Or, Manually Install

1

Download

Grab the latest .dmg from the releases page.

2

Install

Open the .dmg and drag floatpane.app into your /Applications folder.

3

Verify

Enter xattr -cr /Applications/floatpane.app in your terminal to bypass Apple Gatekeeper.
